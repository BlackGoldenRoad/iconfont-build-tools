### 介绍

`iconfont-build-tool` 是一个用来构建 iconfontjs 的工具库，产物引用方式类似 iconfont.cn 的 symbol 模式

### 安装

npm 安装

```html
npm install iconfont-build-tool
```

### 使用示例

请参考example中的 `iconfont.config.js`
